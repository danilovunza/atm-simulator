# ATM Simulator

## Overview

The ATM Simulator is a Python application that simulates basic ATM operations.

Users can:

* Check Balance
* Deposit Money
* Withdraw Money
* Exit the System

The application maintains account balance throughout the session and validates user input.

---

## Features

* Check account balance
* Deposit funds
* Withdraw funds
* Input validation
* Insufficient funds protection
* Interactive menu system
